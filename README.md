# SweepGuard Bot
# Made by 0xSumitro

Recover funds from compromised wallets across EVM chains.

## Free Tier
- Chains: Ethereum, Polygon, Arbitrum, Base

## How to Setup
1. Clone: `git clone https://github.com/xmmitro/sweepguard-bot.git`
2. Install: `npm install`
3. Configure: `.env` file
4. Run: `node bot.js`
